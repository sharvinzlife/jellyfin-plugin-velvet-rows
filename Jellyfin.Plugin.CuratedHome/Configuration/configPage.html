Detection logic

How the shelves decide what to surface.

Newly Added

Ranks the freshest library additions so OCI transfers and recent drops rise to the top immediately.

Newly Released

Uses trusted release metadata only, so old filename-style uploads do not pollute your latest shelves.

Genre discovery

Romance, Thriller, Action, Comedy, Crime, Family, and Mystery shelves lean on Jellyfin genres instead of filenames, then route into dedicated Explore pages.

Rollout notes

Quick reminders while you tune the home screen.

  • Keep Home Screen Sections and Plugin Pages enabled.
  • Use the dedicated Malayalam TV library when available for the cleanest results.
  • Tamil titles in your Malayalam library are intentionally treated as part of the Malayalam shelves.
  • Home shelf headers now point into the matching Velvet Rows Explore page when Plugin Pages is available.
  • If you change My Media pinning or genre shelf publishing, restart Jellyfin once so Home Screen Sections re-registers the updated layout.
