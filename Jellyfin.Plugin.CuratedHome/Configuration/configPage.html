Detection logic

How the shelves decide what to surface.

Newly Added

Ranks the freshest library additions so OCI transfers and recent drops rise to the top immediately.

Newly Released

Prefers premiere dates, but falls back to production year and creation time when metadata is sparse so the row never disappears.

Metadata-aware TV

Malayalam TV rows can search a shared TV library using match terms, while English TV shelves can point straight at your main TV library.

Rollout notes

Quick reminders while you tune the home screen.

  • Use the dedicated Malayalam TV library when available for the cleanest results.
